# Python Type Conversion Project

This project demonstrates different types of **data type conversions** in Python using simple examples.

The program covers:

- Converting user input into different data types
- Working with `int`, `float`, `bool`, and `str`
- Understanding `type()` and `id()`
- Learning how Python stores objects in memory

---

Converts user input into:

- Integer (`int`)
