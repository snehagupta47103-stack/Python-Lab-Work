# Python Conditional Statements Project

## 📌 Overview

This project contains basic Python programs that demonstrate the use of **conditional statements (`if`, `elif`, `else`)**.

The programs included are:

1. Even or Odd Checker
2. Age Category Classifier
3. Largest Number Finder
4. Neutral Number Checker

These programs are beginner-friendly and help in understanding Python decision-making logic.

---

# 📂 Programs Included

## 1️⃣ Even or Odd Checker

### Description

Checks whether a number entered by the user is **Even** or **Odd**.

### Concepts Used

* `if-else`
* Modulus Operator `%`

### Example

```python
Input: 4
Output: Even
```
